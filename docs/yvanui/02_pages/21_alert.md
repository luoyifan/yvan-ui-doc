# 提示框 alert 组件
用于页面中展示重要的提示信息。

![[21_alert/img.png]]


## 事件

| 事件名称   | 说明      | 参数和示例                                |
|:-------|---------|--------------------------------------|
| click  | 获取焦点时触发 |     |
| close  | 关闭时触发   |     |

## 方法
方法一般使用于各类事件中，调用即可生效

| 方法名称 | 说明  | 参数和示例 |
|:-----|-----|-------|
| -    | -   | -     |

## 属性

| 属性名称        | 说明   | 示例  |
|:------------|------|-----|
| type        | 类型   |     |
| title       | 标题   |     |
| description | 描述   |     |
| hidden      | 隐藏   |     |
| showIcon    | 显示图标 |     |
| closable    | 允许关闭 |     |
| closeText   | 关闭文本 |     |
| center      | 文字居中 |     |
