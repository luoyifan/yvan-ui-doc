# 表单 form 组件
高性能表单控件，自带数据域管理，包含输入框（textfield）, 单选框（radiogroup）, 下拉框（combo）等用户输入的组件，可以收集、验证和提交数据。一般以下情况下使用：
- 用于创建一个实体或收集信息。
- 需要对输入的数据类型进行校验时。
## 1. 事件
| 事件名称 | 说明      | 参数和示例                                |
|:-----|---------|--------------------------------------|
| click | 点击 |  |


## 2. 方法
| 方法名称 | 说明         | 参数和示例                                |
|:-----|------------|--------------------------------------|
| trigger | 触发事件 |                              |
| validate | 验证表单 |                              |
| validateField | 验证表单字段 |                              |
| clearValidate | 清理字段的表单验证信息 |                              |
| scrollToField | 滚动到指定的字段 |                              |
| resetFields | 重置表单项 |                              |
| shrinkClick | 触发表单收起和展开事件 |                              |
