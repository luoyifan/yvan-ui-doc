| click | 点击 |  |


## 2. 方法

| 方法名称 | 说明      | 参数和示例                                |
|:-----|---------|--------------------------------------|
| trigger | 触发事件 |                              |
| validate | 验证表单 |                              |
| validateField | 验证字段 |                              |
| clearValidate | 清理字段的表单验证信息 |                              |
| scrollToField | 滚动到指定的字段 |                              |
| resetFields | 重置表单项 |                              |
| shrinkClick | 触发表单收起和展开事件 |                              |
